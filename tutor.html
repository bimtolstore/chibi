Tutorial 1 : Menu musik dapat tetap berjalan dilatar belakang, contoh saat membuka/keluar dari Web ini ke web/apk lain. Tetapi musik akan mati jika membuka halaman lain di Web ini, contoh saat membuka menu Memories Musik akan berhenti

Tutorial 2 : Cara menyalakan musik cukup cari menu musik lali klik buka lalu cari musik yang ingin dinyalakan dan klik tombol play, otomatis musik akan menyala. Dan jika ingin mematikan nya cukup klik tombol play lagi untuk mematikan musiknya atau bisa melalui hapus Task manager pada chrome di tombol kiri di hp chian

Tutorial 3 : Cara menyalakan video di menu memori cukup klik tombol play

Tutorial 4 : Jika memories chat tidak muncul cukup donwload file chatnya lalu buka filenya dan pilih Chrome atau peramban di hp chian!

Tutorial 5 : Jika chian ingin mendownload sesuatu kenangan melalui tombol "Donwload" tetapi tidak tersimpan/unduhan tidak berjalan, maka chian cukup klik titik tiga pada video/musik dan klik donwload
