CHIBI MEMORIES

Tutorial

Ikuti ya diks

Tutorial 1 : Menu musik dapat tetap berjalan dilatar belakang, contoh saat membuka/keluar dari aplikasi ini ke aplikasi. Tetapi musik akan mati jika membuka halaman lain di aplikasi ini, contoh saat membuka menu Memories msuik akan berhenti

Tutorial 2 : Cara menyalakan musik cukup pilih musik dan klik tombol play, otomatis musik akan menyala. Dan jika ingin mematikan nya cukup klik tombol play lagi untuk mematikan musiknya dan bisa melalui hapus Task manager pada tombol kiri di hp anda

Tutorial 3 : Cara menyalakan video di menu memori cukup klik tombol play

Tutorial 4 : Jika memories chat tidak muncul cukup donwload file chatnya lalu buka filenya dan pilih Chrome atau peramban di hp chian!

Tutorial 5 : Jika ingin ke menu utama cukup klik tanda panah ke kiri di pojok kiri atas setiap membuka halaman, Jangan lakukan jika ingin kembali mengklik back pada hp yaitu di kanan paling bawah, itu akan membuat keluar dari aplikasi

Tutorial 6 : Jika chian ingin mendownload sesuatu kenangan tetapi tidak tersimpan/unduhan tidak berjalan, Silahkan chat bimtol untuk masalah ini!

Tutorial 7 : Aplikasi ini pasti akan update terus! dikarenakan masih dalam tahap pengembangan (maklum gue bukan anak IT yg suka bikin apk bjirr, bimtol mah anak Tehnik Mesin)
